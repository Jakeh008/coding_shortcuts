Working With Texts

Always name your html index.html

Use ! to generate the head of the html file (fill in the title)

Headlines - h1-h6 (The bigger the number the smaller the headline)

Bold - strong

Italitcs - em

Notes in code - !-- (shortcut ctrl /)

b and i also bold and italic but never use that

Lists

Unordered lists - use ul to start the list and li for each item

Ordered list - use ol This gives a number, letter, ect before each item

Ordered list with different numbering - ol type="" with a, A, i, I, or 1, inside the quotes

  1. ol type="a"
  2. ol type="a"
  1. type="A"
  2. type="A"
  1. type="i"
  2. type="i"
  1. type="I"
  2. type="I"
  1. type="1"
  2. type="2"

Ordered list from different number - ol start="number" (fill in the number to start at)

  1. Item 1
  2. Item 2

Links

If I want to create a link to an outside source I will use (< a href="link">Title text< /a >)

If I want to create a link to an source inside the website I will use (< a href="html file">Title text< /a >)

If I want the link to open in a new tab I will use (< a href="link" target="_blank" rel="noopener noreferrer">Title text< / a >)

Images

To insert an image from a website use < img src="" alt="" height="310" width="310"> and put the image link or jpeg in the first quotes and a image title in the second quotes
