Working With Texts

Always name your html index.html

For the head of the html just use ! to generate it all. Then fill in the title

I can use h1-h6 for headlines. The bigger the number the smaller the headline

I can bold text using strong and I can use italics with em!

b and i also bold and italic but I should never use that

!-- is used to make notes. Or I can use the shortcut ctrl /

Lists

For unordered lists I will use ul to start the list and li for each item

  • Item 1
  • Item 2
  • Item 3

For an ordered list I will use ol. This gives me a number, letter, ect before each item

If after ol I write type="" with a, A, i, I, or 1, inside the quotes I will get the numbering type I want.

  1. Item 1
  2. Item 2
  1. Item 1
  2. Item 2
  1. Item 1
  2. Item 2
  1. Item 1
  2. Item 2
  1. Item 1
  2. Item 2

I can also use ol start="3" or a different number if i want a continued list

  1. Item 1
  2. Item 2

Links

If I want to create a link to an outside source I will use (< a href="link">Title text< /a >)

If I want to create a link to an source inside the website I will use (< a href="html file">Title text< /a >)

If I want the link to open in a new tab I will use (< a href="link" target="_blank" rel="noopener noreferrer">Title text< / a >)

Images

To insert an image from a website use < img src="" alt="" height="310" width="310"> and put the image link or jpeg in the first quotes and a image title in the second quotes
