Command Line Basics

pwd - prints the name of the working directory

cd - switches into that directory

ls - lists all files and directories in the working directory

cp - copies files or directories. (cp file1 file2 will copy file1 to file2)

cd .. - To move up one directory

mkdir - creates a new directory in the current working directory.

mv - moves a file into a directory or changes file name.

../ - to move backwards

touch - creates a new file inside the working directory.

rm - deletes files

rmdir - deletes directory

rm -r - removes a directory recursively

nano - opens text editor

code - opens VScode

clear - clears the terminal

--help - use after a command for more instrustions on how to use that command

* - matches zero or more characters in a filename, so *.txt matches all files ending in .txt.

? - ? matches any single character in a filename, so ?.txt matches a.txt but not any.txt. (wildcard * and ? explained here)

man - use before a command to show the manual for the specified command
