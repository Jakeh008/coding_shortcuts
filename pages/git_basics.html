Git Basics

Creating a Repository

  1. Create a new repository in Github
  2. Name the repository
  3. Add README file
  4. Get SSH URL
  5. Go to terminal
  6. Type git clone followed by the SSH URL

Git Commands

  • git status - Show the working tree status.
  • git add - Add file contents to the index, to prepare the content staged for the next commit
  • git commit -m - Record changes to the repository
  • git log - show the log of all commits
  • git push origin main - uploads your work to the GitHub

Uploading to the web

Viewing your project on the web
  1. Go to github repository
  2. Click Setting
  3. Click pages
  4. Change branch from none to main and click save
  5. github-username.github.io/github-repo-name (substituting your own details in the link).
